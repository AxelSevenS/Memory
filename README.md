# Project Memory: Pham, Sevenet

## Project setup
```
npm install
```

### Start in Development Mode
```
npm install serve
```
```
npm run serve
```


### Start in Production Mode (with PWA and Service Worker)
```
npm run build
```
```
serve -s dist -l 8080
```
(If serve doesn't work, do this, then try again: ``npm install --global serve``)
